# HS LEWIS FOG - Instruções de Instalação

## Opção 1: Instalar APK Direto (Recomendado)

### No Android:
1. **Baixe o arquivo APK** (`HSLewisApp.apk`)
2. **Abra o arquivo** com o gerenciador de arquivos
3. **Clique em "Instalar"**
4. **Pronto!** O app será instalado automaticamente

### Permissões:
- O Android pode pedir permissões para:
  - Acesso à internet
  - Acesso ao estado da rede
- **Clique em "Permitir"** para ambas

## Opção 2: Compilar Você Mesmo

### Requisitos:
- Android Studio
- Android SDK 24+
- Java 8+

### Passos:
1. Extraia o arquivo `HSLewisApp.zip`
2. Abra em Android Studio
3. Clique em "Run" ou use: `./gradlew assembleDebug`
4. O APK será gerado em: `app/build/outputs/apk/debug/app-debug.apk`
